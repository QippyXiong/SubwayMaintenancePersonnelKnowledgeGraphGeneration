## 维保人员模拟项目

kg: 包含构建的知识图谱本体结构设计（维保人员）
db: Neo4j数据库代码，db目录下设置文件db.json中的信息来连接neo4j数据库，设置的例子如下：
```json
{
    "uri": "neo4j://localhost",
    "auth": ["username", "password"]
}
```
